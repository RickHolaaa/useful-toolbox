# Useful Toolbox

A comprehensive collection of tools designed to simplify daily tasks. All tools run directly in your browser for maximum privacy and speed.

## Features

- **PDF Tools**: Merge, split, and compress PDF documents
- **Image Tools**: Add copyright, resize, remove backgrounds, and convert formats
- More categories coming soon

## Live Demo

Visit [https://rickholaaa.github.io/useful-toolbox/](https://rickholaaa.github.io/useful-toolbox/)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in your browser.

## Build

```bash
npm run build
```

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Deployed on GitHub Pages
